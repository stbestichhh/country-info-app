## Project setup

```bash
$ pnpm install
```

## Compile and run the project

Use [swagger](http://localhost:9180/api) to test sending requests 

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev
```

## Compile and run the project using Docker

```bash
$ docker compose up
```

## Run tests

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e
```

## Documentation

All endpoints have been documented with [Swagger](http://localhost:9180/api).
